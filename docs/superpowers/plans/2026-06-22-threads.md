# Threads Communication Layer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the threads subsystem from the spec (`docs/superpowers/specs/2026-06-20-coaching-profile-nutrition-schema-api-design.md`, "Threads" section): a lightweight coach/client communication layer (`threads` + `thread_messages`) powering a unified inbox, with coach and client REST endpoints.

**Architecture:** Greenfield subsystem following the established `Easy.ClientProfiles` template: a new `Easy.Threads` context over two Ecto schemas (`Easy.Threads.Thread`, `Easy.Threads.ThreadMessage`), coach + client Phoenix controllers (`EasyWeb.Coaches.*` / `EasyWeb.Clients.*`) with OpenApiSpex operations + `CastAndValidate` on writes, and JSON views. Tenant-scoped by `business_id` throughout; a thread belongs to the whole business↔client relationship (any coach in the business sees it; a client sees only their own). No migration to existing tables.

**Tech Stack:** Phoenix controllers + JSON views, OpenApiSpex operations/schemas (`Shared` helpers, `ErrorResponse`), `OpenApiSpex.Plug.CastAndValidate`, Ecto schemas/migration + `Repo.transaction`, ExMachina factories, `Easy.ConnCase` controller tests.

## Global Constraints

Copied from the spec; every task implicitly includes these.

- **Tables** (exact columns):
  - `threads`: id, business_id, client_id, module, subject_type, subject_ref, title, status, priority, last_message_at, last_message_preview, created_by_type, created_by_id, inserted_at, updated_at.
  - `thread_messages`: id, thread_id, author_type, author_id, body, kind, metadata, inserted_at, updated_at. (No `business_id` — messages are scoped through their thread.)
- **Enums:** `module` ∈ {nutrition, training, fitness, profile, general}; thread `status` ∈ {open, resolved, archived}; `priority` ∈ {normal, attention}; `created_by_type`/`author_type` ∈ {coach, client, system}. `subject_type` is a free string (examples: nutrition_week, nutrition_day, nutrition_meal_slot, workout_session, weight_entry, form_submission, client_profile, general); default `general`. `subject_ref` is a map. `kind` (on messages) is a free string; default `message`. `metadata` (on messages) is a map.
- **`author_id`/`created_by_id` is the coach id or client id** (the domain record id, not the user id); null for system. These are TRUSTED — set by the context from the authenticated actor, never from the request body.
- **Ownership rule:** a thread belongs to the business↔client relationship, not to one assigned coach. Any coach in the business can see/act on any client thread (coach controllers scope by `business_id` only). A client can see/act on only their own threads (client controllers resolve the client and scope by `client_id`).
- **Endpoints:**
  - Coach: `GET /v1/coach/threads?client_id=&module=&status=&priority=`, `POST /v1/coach/threads`, `GET /v1/coach/threads/:id`, `PATCH /v1/coach/threads/:id`, `POST /v1/coach/threads/:id/messages`, `GET /v1/coach/clients/:client_id/threads`.
  - Client: `GET /v1/client/threads`, `POST /v1/client/threads`, `GET /v1/client/threads/:id`, `POST /v1/client/threads/:id/messages`.
- **Posting a message updates the parent thread's `last_message_at` and `last_message_preview`** (atomic, in one transaction).
- **Out of scope (do NOT build):** read receipts, reactions, realtime delivery, participants, message editing/deletion. `metadata` is system-only (not exposed in the public message request body).
- `backend/AGENTS.md`: tenant-scope every query by `business_id`; controllers never call `Repo` or build queries; contexts expose verbs + own transactions; trusted IDs set via `put_change`, never `cast`; `@spec` on public functions; no `@moduledoc`/`@doc`; `lib/easy/threads/` holds only Ecto schemas.

## Codebase conventions (verified — match these exactly)

- **Claims:** authenticated requests have `conn.assigns.claims` = `%{user_id, role, business_id, session_id}`. Coach actions read `business_id = conn.assigns.claims.business_id` (and `user_id` when the coach record is needed). Client actions read `%{business_id: business_id, user_id: user_id} = conn.assigns.claims` and resolve the client via `Easy.Clients.get_client_for_user(business_id, user_id) :: {:ok, Client.t()} | {:error, :not_found}`.
- **Coach record:** resolve with `Easy.Coaches.get_by_user_id(business_id, user_id)` → `{:ok, Coach.t()} | {:error, :not_found}` (delegates to `Coach.get_for_user/2`). (If `Easy.Coaches.get_by_user_id/2` is absent, use the private-helper pattern: `Coach |> Coach.for_business(business_id) |> Coach.for_user(user_id) |> Repo.one() |> ok_or_not_found()`.)
- **Controllers:** `use EasyWeb, :controller` + `use OpenApiSpex.ControllerSpecs`; `plug OpenApiSpex.Plug.CastAndValidate, [json_render_error_v2: true] when action in [:create, :update]`; after the plug, read the body from `conn.body_params` and path ids from `conn.path_params["id"]`. `action_fallback` is set in the pipeline (not per-controller).
- **OpenApiSpex:** request schemas pass `struct?: false` as the 2nd arg to `OpenApiSpex.schema/2` and set `additionalProperties: false`. Response entity schemas merge `Shared.timestamps()` into properties. Wrappers: `Shared.data_response(Entity, "Title")` (→ `%{data: ...}`) and `Shared.list_response(Entity, "Title")` (→ `%{data: [...], count: integer}`). `EasyWeb.OpenApi.Schemas.ErrorResponse` exists.
- **FallbackController** already handles `{:error, :not_found}`, `{:error, :unauthorized}`, `{:error, %Ecto.Changeset{}}`, `{:error, %Easy.Error{}}`, and a 500 catch-all. Return those shapes from the context; do not add new clauses.
- **Tests:** `use Easy.ConnCase`; `build_conn() |> authenticate_coach(coach)` / `authenticate_client(client)`; `insert(:factory, overrides)`; `json_response(conn, status)`.
- **Migrations** dir: `backend/priv/repo/migrations/`; latest is `20260621120000_recreate_nutrition_tables.exs`, so use a later timestamp. binary_id PKs; enums as `:string` + DB check constraint + `validate_inclusion`; `:map` → jsonb; `business_id` FK `on_delete: :nothing`; child FKs `on_delete: :delete_all`; polymorphic ids (`created_by_id`, `author_id`) are plain `:binary_id` with no `references`.

defmodule Easy.Threads.Thread do
  use Ecto.Schema

  alias Easy.Clients.Client
  alias Easy.Orgs
  alias Easy.Threads.ThreadMessage

  import Ecto.Changeset
  import Ecto.Query

  @primary_key {:id, :binary_id, autogenerate: true}
  @foreign_key_type :binary_id

  @modules ~w(nutrition training fitness profile general)
  @statuses ~w(open resolved archived)
  @priorities ~w(normal attention)
  @actor_types ~w(coach client system)

  @type t :: %__MODULE__{}

  @spec modules() :: [String.t()]
  def modules, do: @modules

  @spec statuses() :: [String.t()]
  def statuses, do: @statuses

  @spec priorities() :: [String.t()]
  def priorities, do: @priorities

  schema "threads" do
    field :module, :string
    field :subject_type, :string, default: "general"
    field :subject_ref, :map, default: %{}
    field :title, :string
    field :status, :string, default: "open"
    field :priority, :string, default: "normal"
    field :last_message_at, :utc_datetime
    field :last_message_preview, :string
    field :created_by_type, :string
    field :created_by_id, :binary_id

    belongs_to :business, Orgs.Business
    belongs_to :client, Client
    has_many :messages, ThreadMessage

    timestamps(type: :utc_datetime)
  end

  @spec insert_changeset(String.t(), String.t(), %{type: String.t(), id: String.t() | nil}, map()) ::
          Ecto.Changeset.t()
  def insert_changeset(business_id, client_id, creator, attrs) do
    %__MODULE__{}
    |> cast(attrs, [:module, :subject_type, :subject_ref, :title, :priority])
    |> put_change(:business_id, business_id)
    |> put_change(:client_id, client_id)
    |> put_change(:created_by_type, creator.type)
    |> put_change(:created_by_id, creator.id)
    |> validate_required([:module, :business_id, :client_id, :created_by_type])
    |> validate_inclusion(:module, @modules)
    |> validate_inclusion(:priority, @priorities)
    |> validate_inclusion(:created_by_type, @actor_types)
  end

  @spec update_changeset(t(), map()) :: Ecto.Changeset.t()
  def update_changeset(thread, attrs) do
    thread
    |> cast(attrs, [:status, :priority, :title])
    |> validate_inclusion(:status, @statuses)
    |> validate_inclusion(:priority, @priorities)
  end

  @spec for_business(Ecto.Queryable.t(), String.t()) :: Ecto.Query.t()
  def for_business(query \\ __MODULE__, business_id),
    do: from(t in query, where: t.business_id == ^business_id)

  @spec for_client(Ecto.Queryable.t(), String.t()) :: Ecto.Query.t()
  def for_client(query \\ __MODULE__, client_id),
    do: from(t in query, where: t.client_id == ^client_id)

  @spec with_module(Ecto.Queryable.t(), String.t() | nil) :: Ecto.Query.t()
  def with_module(query \\ __MODULE__, module)
  def with_module(query, nil), do: query
  def with_module(query, module), do: from(t in query, where: t.module == ^module)

  @spec with_status(Ecto.Queryable.t(), String.t() | nil) :: Ecto.Query.t()
  def with_status(query \\ __MODULE__, status)
  def with_status(query, nil), do: query
  def with_status(query, status), do: from(t in query, where: t.status == ^status)

  @spec with_priority(Ecto.Queryable.t(), String.t() | nil) :: Ecto.Query.t()
  def with_priority(query \\ __MODULE__, priority)
  def with_priority(query, nil), do: query
  def with_priority(query, priority), do: from(t in query, where: t.priority == ^priority)

  @spec recent(Ecto.Queryable.t()) :: Ecto.Query.t()
  def recent(query \\ __MODULE__) do
    from(t in query, order_by: [desc_nulls_last: t.last_message_at, desc: t.inserted_at])
  end

  @spec with_messages(Ecto.Queryable.t()) :: Ecto.Query.t()
  def with_messages(query \\ __MODULE__) do
    from(t in query, preload: [messages: ^ThreadMessage.ordered()])
  end
end

defmodule Easy.Threads.ThreadMessage do
  use Ecto.Schema

  alias Easy.Threads.Thread

  import Ecto.Changeset
  import Ecto.Query

  @primary_key {:id, :binary_id, autogenerate: true}
  @foreign_key_type :binary_id

  @actor_types ~w(coach client system)

  @type t :: %__MODULE__{}

  schema "thread_messages" do
    field :body, :string
    field :kind, :string, default: "message"
    field :author_type, :string
    field :author_id, :binary_id
    field :metadata, :map, default: %{}

    belongs_to :thread, Thread

    timestamps(type: :utc_datetime)
  end

  @spec insert_changeset(String.t(), %{type: String.t(), id: String.t() | nil}, map()) ::
          Ecto.Changeset.t()
  def insert_changeset(thread_id, author, attrs) do
    %__MODULE__{}
    |> cast(attrs, [:body, :kind])
    |> put_change(:thread_id, thread_id)
    |> put_change(:author_type, author.type)
    |> put_change(:author_id, author.id)
    |> validate_required([:body, :thread_id, :author_type])
    |> validate_inclusion(:author_type, @actor_types)
  end

  @spec for_thread(Ecto.Queryable.t(), String.t()) :: Ecto.Query.t()
  def for_thread(query \\ __MODULE__, thread_id),
    do: from(m in query, where: m.thread_id == ^thread_id)

  @spec ordered(Ecto.Queryable.t()) :: Ecto.Query.t()
  def ordered(query \\ __MODULE__), do: from(m in query, order_by: [asc: m.inserted_at])
end

defmodule Easy.ThreadsTest do
  use Easy.DataCase, async: true

  alias Easy.Threads.Thread
  alias Easy.Threads.ThreadMessage

  @coach %{type: "coach", id: Ecto.UUID.generate()}

  describe "Thread.insert_changeset/4" do
    test "valid with a known module" do
      cs = Thread.insert_changeset("biz", "client", @coach, %{"module" => "nutrition", "title" => "Hi"})
      assert cs.valid?
      assert Ecto.Changeset.get_field(cs, :created_by_type) == "coach"
      assert Ecto.Changeset.get_field(cs, :status) == "open"
    end

    test "rejects an unknown module" do
      cs = Thread.insert_changeset("biz", "client", @coach, %{"module" => "spaceship"})
      refute cs.valid?
      assert "is invalid" in errors_on(cs).module
    end

    test "does not cast business_id/client_id/created_by from attrs" do
      cs =
        Thread.insert_changeset("biz", "client", @coach, %{
          "module" => "general",
          "business_id" => "evil",
          "created_by_id" => "evil"
        })

      assert Ecto.Changeset.get_field(cs, :business_id) == "biz"
      assert Ecto.Changeset.get_field(cs, :created_by_id) == @coach.id
    end
  end

  describe "Thread.update_changeset/2" do
    test "rejects an invalid status" do
      cs = Thread.update_changeset(%Thread{}, %{"status" => "burning"})
      refute cs.valid?
      assert "is invalid" in errors_on(cs).status
    end
  end

  describe "ThreadMessage.insert_changeset/3" do
    test "requires a body" do
      cs = ThreadMessage.insert_changeset("thread", %{type: "client", id: "c1"}, %{})
      refute cs.valid?
      assert "can't be blank" in errors_on(cs).body
    end

    test "sets author from the trusted arg, not attrs" do
      cs = ThreadMessage.insert_changeset("thread", %{type: "client", id: "c1"}, %{"body" => "hey", "author_type" => "system"})
      assert Ecto.Changeset.get_field(cs, :author_type) == "client"
      assert Ecto.Changeset.get_field(cs, :author_id) == "c1"
    end
  end
end

defmodule Easy.Threads do
  alias Easy.Clients
  alias Easy.Clients.Client
  alias Easy.Orgs.Coach
  alias Easy.Repo
  alias Easy.Threads.Thread
  alias Easy.Threads.ThreadMessage

  import Ecto.Query

  @spec list_threads(String.t(), map()) :: {:ok, [Thread.t()]}
  def list_threads(business_id, filters \\ %{}) do
    threads =
      Thread
      |> Thread.for_business(business_id)
      |> maybe_for_client(Map.get(filters, "client_id"))
      |> Thread.with_module(Map.get(filters, "module"))
      |> Thread.with_status(Map.get(filters, "status"))
      |> Thread.with_priority(Map.get(filters, "priority"))
      |> Thread.recent()
      |> Repo.all()

    {:ok, threads}
  end

  @spec list_client_threads(String.t(), String.t()) :: {:ok, [Thread.t()]} | {:error, :not_found}
  def list_client_threads(business_id, client_id) do
    with {:ok, _client} <- get_client(business_id, client_id) do
      list_threads(business_id, %{"client_id" => client_id})
    end
  end

  @spec list_threads_for_user(String.t(), String.t()) :: {:ok, [Thread.t()]} | {:error, :not_found}
  def list_threads_for_user(business_id, user_id) do
    with {:ok, client} <- Clients.get_client_for_user(business_id, user_id) do
      list_threads(business_id, %{"client_id" => client.id})
    end
  end

  @spec get_thread(String.t(), String.t()) :: {:ok, Thread.t()} | {:error, :not_found}
  def get_thread(business_id, thread_id) do
    Thread
    |> Thread.for_business(business_id)
    |> Thread.with_messages()
    |> Repo.get(thread_id)
    |> ok_or_not_found()
  end

  @spec get_thread_for_user(String.t(), String.t(), String.t()) ::
          {:ok, Thread.t()} | {:error, :not_found}
  def get_thread_for_user(business_id, user_id, thread_id) do
    with {:ok, client} <- Clients.get_client_for_user(business_id, user_id) do
      Thread
      |> Thread.for_business(business_id)
      |> Thread.for_client(client.id)
      |> Thread.with_messages()
      |> Repo.get(thread_id)
      |> ok_or_not_found()
    end
  end

  @spec create_thread_as_coach(String.t(), String.t(), map()) ::
          {:ok, Thread.t()} | {:error, :not_found | Ecto.Changeset.t()}
  def create_thread_as_coach(business_id, user_id, attrs) do
    client_id = Map.get(attrs, "client_id")

    with {:ok, coach} <- get_coach(business_id, user_id),
         {:ok, client} <- get_client(business_id, client_id) do
      business_id
      |> Thread.insert_changeset(client.id, %{type: "coach", id: coach.id}, attrs)
      |> Repo.insert()
    end
  end

  @spec create_thread_as_client(String.t(), String.t(), map()) ::
          {:ok, Thread.t()} | {:error, :not_found | Ecto.Changeset.t()}
  def create_thread_as_client(business_id, user_id, attrs) do
    with {:ok, client} <- Clients.get_client_for_user(business_id, user_id) do
      business_id
      |> Thread.insert_changeset(client.id, %{type: "client", id: client.id}, attrs)
      |> Repo.insert()
    end
  end

  @spec update_thread(String.t(), String.t(), map()) ::
          {:ok, Thread.t()} | {:error, :not_found | Ecto.Changeset.t()}
  def update_thread(business_id, thread_id, attrs) do
    with {:ok, thread} <- get_thread_bare(business_id, thread_id) do
      thread
      |> Thread.update_changeset(attrs)
      |> Repo.update()
    end
  end

  @spec add_message_as_coach(String.t(), String.t(), String.t(), map()) ::
          {:ok, ThreadMessage.t()} | {:error, :not_found | Ecto.Changeset.t()}
  def add_message_as_coach(business_id, user_id, thread_id, attrs) do
    with {:ok, coach} <- get_coach(business_id, user_id),
         {:ok, thread} <- get_thread_bare(business_id, thread_id) do
      add_message(thread, %{type: "coach", id: coach.id}, attrs)
    end
  end

  @spec add_message_as_client(String.t(), String.t(), String.t(), map()) ::
          {:ok, ThreadMessage.t()} | {:error, :not_found | Ecto.Changeset.t()}
  def add_message_as_client(business_id, user_id, thread_id, attrs) do
    with {:ok, client} <- Clients.get_client_for_user(business_id, user_id),
         {:ok, thread} <- get_client_thread_bare(business_id, client.id, thread_id) do
      add_message(thread, %{type: "client", id: client.id}, attrs)
    end
  end

  # Private

  defp add_message(thread, author, attrs) do
    Repo.transaction(fn ->
      case ThreadMessage.insert_changeset(thread.id, author, attrs) |> Repo.insert() do
        {:ok, message} ->
          preview = String.slice(message.body, 0, 200)

          Thread
          |> where([t], t.id == ^thread.id)
          |> Repo.update_all(
            set: [
              last_message_at: message.inserted_at,
              last_message_preview: preview,
              updated_at: message.inserted_at
            ]
          )

          message

        {:error, changeset} ->
          Repo.rollback(changeset)
      end
    end)
  end

  defp maybe_for_client(query, nil), do: query
  defp maybe_for_client(query, client_id), do: Thread.for_client(query, client_id)

  defp get_thread_bare(business_id, thread_id) do
    Thread
    |> Thread.for_business(business_id)
    |> Repo.get(thread_id)
    |> ok_or_not_found()
  end

  defp get_client_thread_bare(business_id, client_id, thread_id) do
    Thread
    |> Thread.for_business(business_id)
    |> Thread.for_client(client_id)
    |> Repo.get(thread_id)
    |> ok_or_not_found()
  end

  defp get_client(_business_id, nil), do: {:error, :not_found}

  defp get_client(business_id, client_id) do
    Client
    |> Client.for_business(business_id)
    |> Repo.get(client_id)
    |> ok_or_not_found()
  end

  defp get_coach(business_id, user_id) do
    Coach
    |> Coach.for_business(business_id)
    |> Coach.for_user(user_id)
    |> Repo.one()
    |> ok_or_not_found()
  end

  defp ok_or_not_found(nil), do: {:error, :not_found}
  defp ok_or_not_found(record), do: {:ok, record}
end
