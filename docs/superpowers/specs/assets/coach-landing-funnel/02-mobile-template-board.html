Mobile decisions to keep

First viewport

Show brand, coach positioning, headline, primary CTA, and a hint of the next section. Do not make visitors scroll before understanding the offer.

CTA behavior

Hero CTA is always present. A slim sticky apply bar can appear after scroll, but it must not cover the form or stack with another bottom bar.

Section order

Each template changes order, not components: proof-first moves proof up, problem-fit moves fit up, coach-story moves bio up.
