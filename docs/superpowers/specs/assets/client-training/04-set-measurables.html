Any two-field combo lays out the same way.
One rule covers all 9 types: the block shows 1–2 fields the tracking_type names (reps / weight / time / distance), as ± steppers with tap-to-type. Duration fields add a live hold-timer (▶ count-up → fills the value) so planks and timed carries need zero typing. RPE-based plans show a small RPE field after ✓ ("how hard? 1–10"). The ✓, glow, rest-timer, and carry-forward loop is identical everywhere.
